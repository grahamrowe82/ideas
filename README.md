# ideas.phasetransitions.ai

A warm front door to the Phase Transitions Substack. It frames the writing and sends readers
onward to subscribe; it does not reproduce the pieces.

- Live: https://ideas.phasetransitions.ai
- The publication: https://phasetransitionsai.substack.com

## Structure

- `docs/` — the static site (served by GitHub Pages from `main` `/docs`).
- `DESIGN.md` — the design anchor (quiet scientific / emergence).
- `JOURNEY.md` — the user-journey anchor.
- `journey.steps.json` — the walkthrough the dogfood runner executes.

Built from the Phase Transitions content package and reviewed with the dogfood skill: rendered
and walked headlessly, judged against the design, antislop, and journey anchors.
